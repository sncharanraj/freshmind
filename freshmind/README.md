# 🥬 FreshMind — AI Smart Pantry & Recipe Suggester

## Project Overview
FreshMind helps you track pantry items, get expiry alerts,
and receive AI-powered recipe suggestions.

## Tech Stack
- UI: Streamlit
- Database: SQLite
- AI: Claude API
- Charts: Plotly

## Run the App
```bash
pip install -r requirements.txt
streamlit run app.py
```
